# RoadSoS Web Client
**The Dispatcher's Command Center**

## Overview
A real-time monitoring dashboard for emergency services. Built with Next.js 15, it provides dispatcher awareness through live telemetry streams and AI-driven crash analysis.

## Features
- **Live Stream:** Real-time impact waveforms using Recharts.
- **AI Verdicts:** Instant display of Aegis-Core crash severity analysis.
- **Smart Dispatch:** Dynamic routing alerts showing the nearest trauma center.
- **FSD Architecture:** Clean, modular structure using Feature-Sliced Design.

## Scripts
- `npm run dev`: Launch the dashboard on localhost:3000.
- `npm run build`: Compile for production.
